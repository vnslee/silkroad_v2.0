# 보고서 렌더링 엔진 명세서 (화면 표시)

> **책임 범위:** 생성 엔진이 만든 JSON을 받아 **화면으로 그린다.** 차트 유형 선택·배지·레이아웃·탭 구성.
> **책임 밖:** 산식·조사·채번·데이터 성격 판정 → 생성 엔진 명세서 참조.
> 입력: 생성 엔진 JSON (각 데이터 블록에 `nature`·`source_flag` 포함). 차트 유형은 JSON에 없음 → 이 명세가 결정.

---

## 0. 입력 계약 (생성 엔진 JSON)

각 데이터 블록은 다음 필드를 가진다:
```json
{
  "key": "ev_adoption",
  "label": "EV 보급률",
  "value": [...],
  "nature": "timeseries",      // ← 차트 유형 결정에 사용
  "source_flag": "EXT",        // ← 배지 표시에 사용
  "source": { "org":"...", "date":"...", "tier":"Tier2" }
}
```
> 렌더링은 `value`를 그리고, `nature`로 차트를 고르고, `source_flag`로 배지를 단다.

---

## 1. 데이터 성격 → 차트 매핑 (핵심 규칙)

| `nature` | 기본 차트 | 대안 | 비고 |
|---|---|---|---|
| `timeseries` | **라인차트** | 영역차트 | 연도별 추세. 단일 시점값으로 죽이지 말 것 |
| `composition` | **도넛** | 스택바·파이 | 비율 합=100 |
| `ranking` | **수평 바차트** | 표 | 내림차순 정렬 |
| `score_multiaxis` | **레이더**(단일대상) / **히트맵**(다국×축) | — | 축 3개↑ 레이더, 국가×축 매트릭스는 히트맵 |
| `single_value` | **KPI 카드** | 게이지 | 핵심 수치 강조 |
| `qualitative` | **텍스트 박스** | — | 차트 금지 |
| `status_matrix` | **신호등 매트릭스** | — | 통과=○/탈락=✕, 탈락행 회색 |

**규칙**
- `timeseries`는 반드시 추세 차트(라인/영역). 마지막 한 점만 KPI로 빼는 것 금지.
- 범위값(경쟁사 금리 최저~최고)은 레인지바/도트플롯.
- 2축 비교(매력도×IT유사도)는 산점도, 우상단 강조.
- `value`가 비었거나 "조사 필요"면 **차트 생략 + "조사 필요" 표기**(빈 차트 금지).

---

## 2. 원천 플래그 → 배지 (신뢰도 시각화)

| `source_flag` | 배지 라벨 | 권장 색상 | 의미·주의 |
|---|---|---|---|
| `EXT` | 외부조사 | 회색/뉴트럴 | 출처 있는 사실. 배지 클릭 시 출처·Tier 표시 |
| `INT` | 내부자료 | 파랑 | 캐피탈사 제공값 |
| `CALC` | 계산값 | 초록 | 산식 도출. 재현 가능 |
| `AI` | AI 인사이트 | **보라(강조)** | 추론·검증 필요. 눈에 띄게 |
| `NEWS` | 외부이슈 | **주황(강조)** | 시점 박힌 언론. 출처·날짜 노출 |

**규칙**
- `AI`·`NEWS`는 신뢰도 주의가 필요하므로 **강조색 + 항상 노출**. 묻히지 않게.
- `EXT`·`NEWS`는 배지에 출처 툴팁(기관·날짜·Tier) 필수.
- `CALC`는 호버 시 산식·입력값 추적 표시.
- 한 카드/블록 안에 여러 원천이 섞이면 항목별로 배지를 단다(블록 대표 배지 금지).

---

## 3. 탭·레이아웃 구성

### 유형 1 (단일국가)
탭 순서: **1-0 요약 → 1-1 유사도 → 1-2 결정트리 → 1-3 TCO → 1-4 시장·경쟁배경**
- 1-0 요약: KPI 카드 묶음(유사도·TCO·기간) + AI 인사이트 박스 + 이슈 카드 리스트
- 1-1: 레이더(축별 유사도) + KPI(종합점수·승수)
- 1-2: 결정트리 플로우차트(분기 경로 강조) + (외부솔루션 시)비교표
- 1-3: 워터폴(TCO 분해) + 누적 영역차트(10년) + 스텝차트(구독료 구간) + KPI(총 TCO·기간)
- 1-4: 바(순위) + 레인지바(금리) + 라인(EV추이) + 텍스트(시장특징)

### 유형 2 (권역)
탭 순서: **2-요약 → 2-0 킬스위치 → 2-1 매력도 → 2-2 IT유사도/순위 → 2-3 시장배경**
- 2-요약: 순위 KPI(1~3위) + AI 인사이트 박스 + 이슈 카드 리스트
- 2-0: 신호등 매트릭스(국가×킬스위치)
- 2-1: 수평바(매력도 순위) + 스택바(기여분) + 라인(GDP·CAGR)
- 2-2: 히트맵(국가×축) + 순위표/바 + 산점도 + **국가 프로파일 카드(상위 3개국, 나란히)**
- 2-3: 바(브랜드·OEM) + 도넛(구매유형·세그먼트)

### 국가 프로파일 카드 (유형2, 상위 3개국)
- 순위표 **아래**에 1·2·3위 카드 **나란히**(비교 가능). 탭 분리 금지.
- 카드 1장: 순위·국가·퀵윈점수 / 시장 한 줄 / 경쟁 상황 / 킬스위치 상태 / 핵심 이슈 1건 / AI 코멘트
- 각 줄 끝에 원천 배지(§2). 4위 이하는 순위표에만.

## 4. 공통 렌더링 원칙
- 요약 탭이 **맨 앞**(의사결정자용). 세부 탭은 뒤로.
- 시계열은 추세가 보이게(최소 3~5개년 축). 데이터가 단일 시점뿐이면 KPI로 대체 + "추세 데이터 없음" 표기.
- 모든 수치 옆 원천 배지 필수(근거 추적). 배지 없는 수치 = 렌더링 오류로 간주.
- "조사 필요" 항목은 빈 차트 대신 명시적 플레이스홀더.
- 유형2 순위는 10점 구간 표기(과잉정밀 금지) — 1점 차 강조 금지.
- **통화 표시:** 금액은 `KRW 환산값 + (현지통화 원값)` 병기. 예: `₩3,200만 (PLN 92,000)`. 보고서 어딘가에 환율 기준일(fx_date) 1회 표기. 유형2 다국 비교 차트는 전부 KRW 축으로 통일.

## 5. 탭별 차트 상세 (생성 JSON → 렌더링 매핑)

> 각 탭의 데이터 블록을 어떤 차트로 그리는지 상세. 생성 JSON의 `nature`·`source`를 소비.

### 유형 1
**[탭 1-0 요약]**
- **KPI 카드 묶음** — 유사도·TCO·기간·시스템결정.
- **이슈 카드 리스트** — 이슈별 한 줄 + 함의 + 출처배지. (차트 아님)

**[탭 1-1 유사도]**
- **레이더차트** — 축=4개 유사도축(시스템/상품/규제/리스크), 값=축별 유사도점수(0~100), 계열=A국 vs B국(또는 A국 단일). 소스=축별 유사도. → "어느 축이 닮고 어디가 갈리나" 한눈에.
- **KPI 카드** — 종합 유사도 점수(0~100) + 그에 따른 적용 승수%(탭1-3 연동). 소스=종합 유사도.

**[탭 1-2 결정트리]**
- **결정트리 다이어그램(플로우차트)** — 유사도 점수가 어느 분기(70↑/50미만/그외)로 흘러 어떤 결론(B확산/외부솔루션/본사구축)에 도달했는지 경로 강조. 소스=탭1-1 종합점수 + 분기 임계값.
- (외부솔루션 분기 시) **비교표** — 추천 솔루션 2~3종 × A국 요건 커버리지. 차트 아님, 표로 출력.

**[탭 1-3 TCO]**
- **워터폴 차트** — TCO 구성 분해: 구축비 → +특화개발비 → +유지비(10년) → +운영비 → 총 TCO. 소스=산식4. → 어느 항목이 비용을 끌어올리나.
- **누적 라인/스택 영역차트(10년)** — X=연차(1~10년), Y=누적비용. 계열=일회성(구축+특화) vs 반복(유지+운영). 소스=산식4 연차전개. ◷시계열 출력.
- **스텝 차트(구독료 구간)** — X=누적 계약건수, Y=건당 단가. 신규 진출로 구간이 어떻게 내려가는지(소급) 표시. 소스=구독료 구간표[INT] + 산식3.
- **KPI 카드** — 총 10년 TCO, 예상 구축기간, A국 예상 계약건수.

**[탭 1-4 시장·경쟁 배경]**
- **바차트(순위)** — 금융사 순위, OEM 순위. X=사명, Y=점유율·순위. 정태.
- **레인지 바/도트플롯** — 경쟁사 금리범위(최저~최고). 소스=경쟁사 금리.
- **라인차트(시계열)** — EV 보급률 추이, EV·ICE 잔존가치 추이. X=연도, Y=%. ◷시계열 출력 → 리스 손익 추세.
- **KPI 카드** — 평균 신차가격. **텍스트 박스** — 시장특징 정성요약(차트 아님).

### 유형 2
**[탭 2-요약]**
- **순위 KPI 카드** — 1~3위국 + 점수.
- **이슈 카드 리스트** — 권역/국가별 이슈 + 함의 + 출처배지.

**[탭 2-0 킬스위치]**
- **통과/탈락 매트릭스(신호등 표)** — 행=권역 내 국가, 열=킬스위치 4종, 셀=통과(○)/탈락(✕). 탈락국은 회색 처리. 소스=킬스위치 항목. → 어느 나라가 무엇 때문에 걸렸나 한눈에.

**[탭 2-1 매력도]**
- **수평 바차트(순위)** — 권역 내 국가별 매력도 점수(0~100) 내림차순. 소스=산식2. → 매력도 줄세우기.
- **스택 바/누적막대** — 국가별 점수를 항목 기여분(GDP·시장규모·침투율·경쟁강도·계약방식)으로 분해. 소스=항목 정규화점수. → 어느 항목이 점수를 끌었나.
- **라인차트(시계열)** — GDP 성장률·시장 CAGR 추이(후보국 비교). X=연도, Y=%. ◷시계열 출력.
- **버블차트(선택)** — X=시장규모, Y=금융침투율, 버블=판매대수. 시장 포지셔닝.

**[탭 2-2 IT유사도/순위]**
- **히트맵** — 행=권역 내 국가, 열=4개 축(시스템/규제/상품/회수), 셀=밴드(상/중/하) 색상. 소스=밴드 채점. → 어느 나라가 어느 축에서 B와 닮았나.
- **최종 퀵윈 순위표/바차트** — 매력도(2-1)+IT유사도(2-2) 합산 순위. 10점 구간 표기(과잉정밀 금지). 소스=최종 퀵윈 점수.
- **2축 산점도(선택)** — X=비즈니스 매력도, Y=IT 유사도. 우상단=퀵윈 최적국. → 비즈니스·IT 동시 우수국 식별.

**[탭 2-3 시장 배경]**
- **바차트(순위)** — 브랜드 Top10, OEM 순위. 정태.
- **도넛/파이** — 구매유형 비중(승용:이륜:상용), 고객 세그먼트(개인:법인). 구성비.
- **KPI 카드** — 평균 구매주기(년). **텍스트** — 화이트스페이스 세그먼트(정성).

## 6. 종합 시각화 명세표

> 전체 차트를 한눈에. 생성 JSON의 데이터성격(nature)별로 차트가 결정된다.

| 탭 | 차트유형 | X축 / 분류 | Y축 / 값 | 데이터소스 | 데이터성격 |
|---|---|---|---|---|---|
| 1-0 | KPI카드+이슈리스트 | — | 유사도·TCO·기간·이슈 | 타 탭 + 웹조사 | 요약 |
| 1-1 | 레이더 | 4개 유사도축 | 유사도(0~100) | 축별 유사도 | 정태(스냅샷) |
| 1-1 | KPI카드 | — | 종합점수·승수% | 종합 유사도 | 정태 |
| 1-2 | 플로우차트 | 분기경로 | — | 종합점수+임계값 | 정태(로직) |
| 1-3 | 워터폴 | TCO 구성요소 | 비용 | 산식4 | 정태(스냅샷) |
| 1-3 | 누적 스택영역 | 연차 1~10 | 누적비용 | 산식4 연차전개 | ◷시계열 |
| 1-3 | 스텝차트 | 누적 계약건수 | 건당단가 | 구독료 구간표 | 정태(구간) |
| 1-4 | 바(순위) | 사명 | 점유율 | 금융사·OEM 순위 | 정태 |
| 1-4 | 레인지바 | 경쟁사 | 금리 최저~최고 | 경쟁사 금리 | ◷시계열성 |
| 1-4 | 라인 | 연도 | EV보급률·잔가(%) | EV 보급·잔가 | ◷시계열 |
| 2-요약 | KPI카드+이슈리스트 | — | 순위·이유·이슈 | 타 탭 + 웹조사 | 요약 |
| 2-0 | 신호등 매트릭스 | 국가×킬스위치 | 통과/탈락 | 킬스위치 4종 | ⚡이벤트 |
| 2-1 | 수평바(순위) | 국가 | 매력도(0~100) | 산식2 | 정태(스냅샷) |
| 2-1 | 스택바 | 국가 | 항목 기여분 | 항목 정규화점수 | 정태 |
| 2-1 | 라인 | 연도 | GDP·CAGR(%) | 성장률·CAGR | ◷시계열 |
| 2-2 | 히트맵 | 국가×4축 | 밴드(상/중/하) | 밴드 채점 | 정태 |
| 2-2 | 산점도 | 매력도 | IT유사도 | 2-1+2-2 | 정태 |
| 2-2 | 순위표/바 | 국가 | 퀵윈 점수(구간) | 최종 합산 | 정태 |
| 2-2 | 국가 프로파일 카드 | 상위 3개국 | 요약(점수·시장·이슈) | 타 탭 종합 | 요약 |
| 2-3 | 바(순위) | 브랜드·OEM | 점유율 | 순위 | 정태 |
| 2-3 | 도넛 | 구매유형·세그먼트 | 비중 | 구성비 | 정태 |
